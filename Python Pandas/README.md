# Pandas Practice Notebooks
This repository contains **11 beginner-friendly Jupyter Notebooks** to help you learn **Python Pandas** step by step. Each notebook explains one topic using simple examples that are easy to understand.
## Topics Covered
### Video 1 – Introduction to Pandas
- Install and import Pandas
- Create a DataFrame
- Load a CSV file
- View dataset information
### Video 2 – DataFrame and Series
- Create Series and DataFrames
- Select rows and columns
- Use `loc` and `iloc`
### Video 3 – Indexing
- Set an index
- Reset an index
- Find data using indexes
### Video 4 – Filtering Data
- Filter rows using conditions
- Use comparison operators
- Combine multiple conditions
### Video 5 – Updating Data
- Rename columns
- Update rows and columns
- Use `apply()`, `map()`, and `replace()`
### Video 6 – Adding and Removing Data
- Add new columns
- Delete columns
- Add new rows
- Remove rows
### Video 7 – Sorting Data
- Sort rows and columns
- Sort by index
- Find highest and lowest values
### Video 8 – Grouping Data
- Use `groupby()`
- Find mean, count, and other summaries
- Analyze grouped data
### Video 9 – Cleaning Data
- Find missing values
- Fill or remove missing values
- Change data types
- Clean datasets
### Video 10 – Dates and Time
- Work with dates
- Convert columns to datetime
- Filter by dates
- Resample time series data
### Mini Project: Sales Data Analysis
- Explore a sales dataset
- Find total sales and profit
- Analyze products and categories
- Create simple summaries
- Practice real-world data analysis
## Requirements
Install the required libraries:
```bash
pip install pandas jupyter
```
## How to Use
1. Download or clone this repository.
2. Install the required libraries.
3. Open Jupyter Notebook.
4. Open any notebook (`Video_01.ipynb` to `Video_11.ipynb`).
5. Run each cell one by one and practice.
## What You Will Learn
After completing these notebooks, you will be able to:
- Load datasets
- Work with DataFrames and Series
- Filter and update data
- Add and remove rows and columns
- Sort and group data
- Clean datasets
- Work with dates and time
- Analyze real-world sales data
- Build a strong foundation in Pandas
