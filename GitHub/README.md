# Git Cheat Sheet
## About This Project
This project contains a simple **Git Cheat Sheet** that covers the most commonly used Git commands. It is designed for beginners who want a quick reference while learning Git and GitHub.
# Topics Covered
The cheat sheet includes the following topics.
## 1. Installation
Learn how to install Git on different operating systems.
Includes:
- Git for Windows
- Git for Mac
- Git for Linux
- Official Git website
## 2. Git Setup
Learn how to configure Git before using it.
Topics include:
- Setting username
- Setting email
- Enabling colored output
Common commands:
- `git config --global user.name`
- `git config --global user.email`
- `git config --global color.ui auto`
## 3. Initialize and Clone Repository
Learn how to create or copy a Git repository.
Topics include:
- Initialize a repository
- Clone an existing repository
Commands:
- `git init`
- `git clone`
## 4. Staging and Committing
Learn how to save your project changes.
Topics include:
- Check repository status
- Stage files
- Unstage files
- View changes
- Commit changes
Commands:
- `git status`
- `git add`
- `git reset`
- `git diff`
- `git commit`
## 5. Branching and Merging
Learn how to work with branches.
Topics include:
- Create branches
- Switch branches
- Merge branches
- View branch history
Commands:
- `git branch`
- `git checkout`
- `git merge`
- `git log`
## 6. Inspect and Compare
Learn how to check commit history and compare changes.
Topics include:
- View commit history
- Compare branches
- View file history
- Show commit details
Commands:
- `git log`
- `git diff`
- `git show`
## 7. Share and Update
Learn how to connect with remote repositories.
Topics include:
- Add remote repository
- Fetch updates
- Pull latest changes
- Push local changes
Commands:
- `git remote add`
- `git fetch`
- `git pull`
- `git push`
## 8. Track File Changes
Learn how to rename or remove files.
Topics include:
- Delete files
- Rename files
- View moved files
Commands:
- `git rm`
- `git mv`
- `git log --stat -M`
## 9. Rewrite History
Learn how to modify Git history.
Topics include:
- Rebase branches
- Reset commits
Commands:
- `git rebase`
- `git reset --hard`
## 10. Ignore Files
Learn how to prevent files from being tracked by Git.
Topics include:
- Create `.gitignore`
- Ignore files and folders
- Global ignore rules
Command:
- `git config --global core.excludesfile`
## 11. Git Stash
Learn how to temporarily save your work.
Topics include:
- Save changes
- View stashed changes
- Restore changes
- Delete stash
Commands:
- `git stash`
- `git stash list`
- `git stash pop`
- `git stash drop`
# What You Will Learn
After reading this cheat sheet, you will be able to:
- Install Git
- Configure Git
- Create and clone repositories
- Track project changes
- Commit your work
- Create and manage branches
- Merge branches
- Push and pull code using GitHub
- Compare project versions
- Ignore unnecessary files
- Temporarily save your work using Git Stash
# Requirements
To use this cheat sheet, you need:
- Git installed on your computer
- A GitHub account (optional)
- Basic knowledge of the command line
# How to Use
1. Install Git on your computer.
2. Configure your username and email.
3. Create or clone a repository.
4. Follow the commands in the cheat sheet while working on your projects.
5. Use it as a quick reference whenever needed.
