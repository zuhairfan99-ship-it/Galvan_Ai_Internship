# Python Mini Projects
## About
This repository contains 5 simple Python mini projects. These projects were created to practice Python programming concepts like variables, loops, conditions, functions, lists, files, and user input.
# Project 1: Choose Your Own Adventure Game
## Description
This is a simple text-based adventure game. The player makes different choices, and each choice changes the story. The game ends with either a win or a loss.
## Features
- Text-based gameplay
- Multiple choices
- Different endings
- Easy to play
## Concepts Used
- if-else
- User input
- Strings
- Nested conditions
# Project 2: Text-Based Calculator
## Description
This calculator performs basic math operations. The user enters two numbers and chooses an operation.
## Features
- Addition
- Subtraction
- Multiplication
- Division
- Error handling for invalid input
- Division by zero check
## Concepts Used
- Functions
- Loops
- Exception handling
- User input
# Project 3: Contact Management System
## Description
This project stores contact information. Users can add, view, search, and delete contacts. The contacts are saved in a JSON file.
## Features
- Add contact
- View contacts
- Search contact
- Delete contact
- Save data in JSON file
## Concepts Used
- Lists
- Dictionaries
- Functions
- JSON file handling
- Loops
# Project 4: Tic-Tac-Toe with AI
## Description
This is a Tic-Tac-Toe game where the user plays against the computer. The computer uses AI to make smart moves.
## Features
- Human vs AI
- Smart AI moves
- Win, lose, and draw detection
- Easy interface
## Concepts Used
- Functions
- Lists
- Loops
- Conditions
- Minimax Algorithm
# Project 5: Automated Quiz Generator
## Description
This project creates a quiz using random questions. At the end, it shows the score and the time taken.
## Features
- Random questions
- Multiple-choice answers
- Score calculation
- Time tracking
- Quiz summary
## Concepts Used
- Lists
- Dictionaries
- Random module
- Time module
- Loops
# Requirements
- Python 3.14.6
# How to Run
1. Download the project files.
2. Open Terminal or Command Prompt.
3. Go to the project folder.
4. Run the file using:
```bash
python filename.py
```
Example:
```bash
python Project1.py
```
