# Python Practice Notebook (Videos 1–9)
## About This Project
This project is a **Python practice notebook** created while following a Python tutorial series. It contains examples and hands-on practice for important Python concepts, from strings and loops to file handling and working with dates and times.
# Topics Covered
This notebook covers the following topics.
# 1. Strings
Learn how to work with strings in Python.
### Topics include:
- Creating strings
- String length
- Indexing
- Slicing
- Converting to uppercase and lowercase
- Finding and replacing text
- String methods
- String concatenation
- f-strings
- String formatting
# 2. Conditional Statements
Learn how to make decisions in Python.
### Topics include:
- `if`
- `elif`
- `else`
- Comparison operators
- Logical operators (`and`, `or`, `not`)
- Boolean values
# 3. Identity and Boolean Values
Understand how Python compares objects.
### Topics include:
- `id()` function
- Object identity
- True and False values
- Empty sequences
- Empty dictionaries
- `None`
# 4. Loops
Learn how to repeat tasks automatically.
### Topics include:
- `for` loops
- Nested loops
- `while` loops
- `break`
- `continue`
- `range()` function
# 5. String Formatting
Learn different ways to format text.
### Topics include:
- `.format()`
- f-strings
- Formatting numbers
- Formatting dictionaries
- Formatting objects
- Formatting dates
- Decimal formatting
# 6. Classes and Objects
Learn the basics of Object-Oriented Programming (OOP).
### Topics include:
- Creating classes
- Constructors (`__init__`)
- Creating objects
- Accessing object attributes
# 7. Date and Time
Learn how to work with dates and times using Python.
### Topics include:
- Current date
- Current time
- Date arithmetic
- Time differences
- `datetime` module
- `timedelta`
- `strftime()`

Learn how to read and write files.

- Opening files

- Using `with` statement
