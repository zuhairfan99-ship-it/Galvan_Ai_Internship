# Project 1: Student Grade Management System
## About the Project
This project is a simple Student Grade Management System built using Python.
It helps you store student information, calculate their total marks, percentage, and grade automatically. The project is beginner-friendly and uses simple Python concepts like lists, dictionaries, functions, loops, and conditions.
## Features
- Add student details
- Store marks of multiple subjects
- Calculate total marks
- Calculate percentage
- Assign grades automatically
- Display all student records
- Save data in a Pandas DataFrame
## Technologies Used
- Python
- Pandas
## Functions Included
### add_student()
Adds a new student and their marks.
### calculate_total()
Calculates the total marks.
### calculate_percentage()
Calculates the percentage based on total marks.
### assign_grade()
Assigns a grade according to the percentage.
Example:
- 90 and above → A+
- 80–89 → A
- 70–79 → B
- 60–69 → C
- 50–59 → D
- Below 50 → F
### process_student()
Creates the complete student record with total, percentage, and grade.
### display_students()
Shows all student records.
## Output
The final output shows:
- Student Name
- Marks
- Total Marks
- Percentage
- Grade
## Learning Objectives
This project helps beginners learn:
- Functions
- Lists
- Dictionaries
- Loops
- Conditional Statements
- Basic Data Processing
- Pandas DataFrame
## Future Improvements
- Save records in CSV
- Search student by name
- Delete student
- Update marks
- Build a graphical interface
